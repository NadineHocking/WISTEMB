# WISTEM
A multipage Website for Women in STEM Conference

## A simple, responsive multi-page website build using HTML, and CSS Flexbox

# The website consists of:

- 1.index.htm
- 2.about.htm
- 3.speakers.htm
- 4.schedule.htm
- 5.tickets.htm

# Features

- Responsive, desktop-first design
- CSS flexbox for layout, alignment and position 
- Images
- Custom logo
- Internal links
- Variables 
- Light and dark theme variables
- Contact page and styled form
- Pseudo-elements and classes
