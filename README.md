# WISTEMB
A multipage Website for Women in STEM Conference

## A simple, responsive multi-page website build using HTML, CSS and Bootstrap

# The website consists of:

- 1.index.htm
- 2.about.htm
- 3.speakers.htm
- 4.schedule.htm
- 5.tickets.htm

# Features

- Responsive, desktop-first design
- Bootstrap for layout, alignment and position 
- Images
- Custom logo
- Internal links
- Variables 
- Light and dark theme variables
- Contact page and styled form
- Pseudo-elements and classes

# Bootstrap implementation and updated styles

- Bootstrap navigation with active classes and toggle button for small devices
- Added Bootstrap js bundle and jQuery to enable navigation toggle function
- Updated columns, spacing and alignment with Bootstrap, preset styles
- Updated Speakers gallery with Bootstrap card layout
